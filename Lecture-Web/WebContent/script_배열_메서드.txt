배열에 엘리먼트 추가/삭제하기

- .push() : 배열의 뒤에 엘리먼트 추가
- .pop() : 배열의 뒤에서 엘리먼트 삭제하고 리턴
- .shift() : 배열의 앞에서 엘리먼트 삭제하고 리턴
- .unshift() : 배열의 앞에 엘리먼트 추가

배열 붙이기, 검색하기(concat은  String도 가능)
- .arr1.concat(arr2) : arr과 arr2 붙임
  => 추가로 연결하게 된다면 : .arr1.concat(arr2).concat(arr3)
- arr.indexOf(element) : arr에서 element가 있는 첫 위치를 검색
- arr.lastIndexOf(element) : arr에서 element가 있는 마지막 위치를 검색 

문자열 split 함수
- 문자열을 구분자(separator)로 나눠서 각각을 담은 배열을 반환하는 함수 

그 외 
 - arr.reverse() : 배열을 순서를 역순으로 바꾼다.
 - 구분자 => arr.sort() : 배열 안의 값들을 바르게 정렬해준다.
